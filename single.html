San Francisco Traffic Collision Severity Prediction Using Advanced Machine Learning (*On-Going Project*)

This project leverages advanced machine learning techniques to predict the severity of traffic collisions in San Francisco.

January 25, 2024 Jino anbu

The goal of this assignment is to explore and analyze the Traffic Crash resulting in Injury Dataset and perform clustering, outlier detection and Classification using various methods and analyze the performance of the various algorithms and find the best performing algorithm on this dataset.

Utilizing a dataset spanning 17 years (2005-2022), the project applies various clustering, outlier detection, and classification models to analyze injury severity and contributing factors. Key techniques include K-means clustering, DBSCAN, Expectation-Maximization, Local Outlier Factor, and Random Forest, among others. The goal is to support San Francisco's Vision Zero initiative to eliminate traffic fatalities and severe injuries by 2024. This comprehensive study not only predicts injury severity but also identifies crucial contributing factors, aiding in effective strategic planning for improved road safety.
